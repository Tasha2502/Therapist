Madhav 7.0 – Therapist Chatbot

Madhav 7.0 is a conversational AI application designed to provide a calming and supportive environment for individuals seeking emotional support, guidance, and mindfulness exercises. It acts as a virtual therapist, offering empathetic responses, stress-relief tips, and friendly conversation for mental wellness.

Features
	•	Real-time chat interface with light-dark calming theme inspired by nature (mountains, rivers, greenery).
	•	Empathetic responses to loneliness, anxiety, and emotional queries.
	•	Mindfulness prompts and stress-relief suggestions.
	•	Responsive design for desktop and mobile.
	•	Lightweight frontend using React.js and Tailwind CSS.
	•	Backend powered by Node.js and Express.js for message handling.

Tech Stack
	•	Frontend: React.js, Tailwind CSS
	•	Backend: Node.js, Express.js
